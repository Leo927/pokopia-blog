## 概述

自第五世代以来，雨天队一直是对战中的经典天气流派。大嘴鸥的降雨特性可以自动召唤雨天，配合悠游自如的高速打手，雨天队始终是最稳定的天气体系之一。

## 核心成员

### 天气手：大嘴鸥（Pelipper）

大嘴鸥是 VGC 中最优秀的雨天召唤者。降雨特性在上场时自动生效，同时还能提供丰富的辅助招式。

```
大嘴鸥 @ 气势披带
特性：降雨
努力值：4 HP / 252 特攻 / 252 速度
性格：胆小
- 气象球
- 暴风
- 顺风
- 守住
```

### 悠游自如打手：梭子鱼（Barraskewda）

雨天状态下，梭子鱼成为场上最快的宝可梦之一。

```
梭子鱼 @ 神秘水滴
特性：悠游自如
努力值：252 攻击 / 4 特防 / 252 速度
性格：固执
- 水流裂破
- 近身战
- 水流喷射
- 守住
```

## 作战思路

1. **首发大嘴鸥 + 打手** — 立刻召唤雨天并施加压力
2. **顺风作为备选** — 如果雨天被覆盖，顺风能保持速度优势
3. **积极使用守住** — 消耗对方天气回合
4. **灵活换人** — 需要时重新触发降雨

## 常见威胁与对策

| 威胁 | 对策 |
|------|------|
| 煤炭龟（晴天） | 保持大嘴鸥血量以重新召雨 |
| 草系宝可梦 | 大嘴鸥使用暴风 |
| 避雷针特性 | 集火另一只宝可梦 |

## 结语

雨天队奖励激进的打法和精准的站位。多练习残局处理——知道何时牺牲大嘴鸥、何时保留它，是普通雨天玩家和顶尖雨天玩家的分水岭。
